# React-native-metal-tracker
A React Native mobile application that displays live prices of precious metals including Gold, Silver, Platinum, and Palladium. The app provides a clean financial dashboard experience with real-time data, detailed insights, and smooth navigation.
### Requirements:
#### 1. App Setup
- Initialize a React Native project
- Use clean folder structure
- Ensure proper navigation setup
#### 2. Features
- Display live prices for:
    - Gold
    -  Silver
    -  Platinum
    -  Palladium
- Show:
     - Current price
     - Price change (increase/decrease)
     - Timestamp of last update
#### 3. UI/UX
- Clean and modern financial dashboard UI
- Use cards or tiles for each metal
- Add icons/images for each metal
- Responsive design
#### 4. API Integration
- Fetch real-time data using any public API
- Handle loading and error states
#### 5. Extra Features (Optional)
- Graphs or charts for price trends
- Dark/light mode
- Refresh button
### Tech Stack:
- React Native
- JavaScript / TypeScript
- REST APIs
